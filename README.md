# CampusPYQ

A full-stack web application for managing Previous Year Question Papers.

## Features
- Admin Login
- Branch Management
- Semester Management
- Subject Management
- Upload PDF
- View PDF
- Download PDF
- Search
- Filter
- Change Password

## Tech Stack
Frontend
- React
- Axios
- React Router
- React Toastify

Backend
- Spring Boot
- Spring Security
- Spring Data JPA
- Hibernate

Database
- MySQL
